# sber_test_api


Развёртывание приложения
========================

### Сборка
    docker build -t sber_tz .

### Запуск
    docker run -p 8080:8080 sber_tz

Тесты
========================
    pytest --cov=app app/tests/

Запускает тесты и показывает покрытие кода


Входные данные
========================
Формат входных данных указан в `entry_data_schema.json`
